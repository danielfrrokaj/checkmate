Privacy Policy

Last Updated: March 2025

1. Introduction

Focus is a Chrome extension designed to help users manage their tasks and track time efficiently. This Privacy Policy explains how we handle user data.

2. Data Collection and Usage

CheckMate does not collect, store, or transmit any personal or sensitive user data. All tasks and timer settings are stored locally on the user's device using Chrome's local storage API (chrome.storage.local).

3. Data Sharing

CheckMate does not share any user data with third parties, external services, or remote servers. All data remains on the user's device.

4. Security

Since all data is stored locally within the browser, CheckMate does not transmit any information over the internet, reducing privacy risks. Users have full control over their stored data and can clear it at any time by removing the extension or clearing browser storage.

5. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated date.
